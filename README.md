# 🌌 RESOnation — Community Platform

**Style:** Tech-Zen (Oxytocin on a steel frame).  
**Target Audience:** Impact-driven entrepreneurs, expats, and conscious individuals.  
**Mission:** Decoding the Social Code to find your tribe.

## 🎨 Design Concept
- **Base:** Deep Space Gray (`#1A1A1B`)
- **Signal Blue:** Electric Blue (`#007AFF`)
- **Oxytocin Orange:** Sunset Copper (`#FF8C42`)

## ✨ Key Features
- **Social Code Interface**: A unique UI for connecting with like-minded people.
- **Architect Section**: Story of the community founder with high-fidelity visuals.
- **Live Moments Gallery**: Draggable photo strip showcasing high-energy community events.
- **Multiplier Technology**: A grid-based showcase of community values and tools.

## 🛠 Tech Stack
- **Languages**: HTML5, Vanilla JavaScript, CSS3
- **Aesthetics**: Glassmorphism, smooth animations, editorial-style imagery.
